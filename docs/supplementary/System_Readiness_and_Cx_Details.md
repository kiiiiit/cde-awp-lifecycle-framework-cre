# Системная готовность и ПНР (детали)

## MC‑пакеты
Перечень по системам МЭП/ядра/фасадов; формула перевода объёмов в % готовности систем.

## Матрица испытаний
Автономные, функциональные, интеграционные; артефакты: протоколы, акты, журналы несоответствий.

## KPI блока ПНР
Доля успешных испытаний с первого раза, среднее время устранения, частота повторных тестов, влияние на критический путь.
